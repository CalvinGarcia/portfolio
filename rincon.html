Rincon Beauty

Description

Rincon beauty is a fictional beauty brand with a natural eco friendly focus. This brand standards guide was created to show comprehension of color, typography, and how it can be utilized to convey a message.

This project was created primarily using Adobe Indesign, with assets created by myself in Adobe Illustrator. I chose to create a brand standards guide based on the beauty industry because I wanted to test my capacity to create for a market that I myself am not part of.

2020 Calvin Garcia
